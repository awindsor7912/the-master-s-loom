# The Master's Loom
An AI-augmented TTRPG Command Center built on the Microsoft Power Platform. Featuring real-time Dataverse synchronization, Power Automate-driven session orchestration, and Azure OpenAI integration for dynamic narrative weaving and automated session chronicles.

## Overview
The Master's Loom is an interactive operational platform designed to eliminate paper-heavy tracking and manual data silos in tabletop gaming. Built with a focus on real-time synchronization and AI-driven narrative assistance, the platform functions as a "Single Source of Truth" for campaigns, character progression, and live session management.

---

## 🚀 Key Capabilities

### 🏰 Campaign Architecture
* **Intake Wizard:** Guided workflow for campaign creation, setting rulesets, and world-building.
* **Smart Import:** AI-driven document processing to ingest existing campaign PDFs and notes into structured data.
* **AI Map Generation:** On-demand generation of tactical maps and location descriptions.

### ⚔️ Live Session Command (HUD)
* **Real-Time Combat Monitor:** A live dashboard for DMs to view player vitals (HP, AC, Stats) as players update them in real-time.
* **Multimodal Session Logging:** Live transcription of game sessions with automated AI-generated "Story So Far" summaries.
* **Secret Communications:** Encrypted 1-on-1 "Whisper" chats between the DM and individual players.
* **Interactive Battlemap:** Dynamic map sharing with digital miniatures that sync across all player devices.

### 📜 Character Mastery
* **Live Character Sheets:** Interactive sheets that sync directly with the DM’s dashboard.
* **Digital Figurines:** AI-generated or user-uploaded character icons for tactical map placement.
* **Personal Chronicles:** Private note-taking for players with historical access across sessions.

## 🛠️ Technical Stack (Proposed)
* **Logic Engine:** Power Automate
* **Data Layer:** Microsoft Dataverse
* **Interface:** Power Apps (Canvas)
* **Intelligence:** Azure OpenAI / AI Builder
